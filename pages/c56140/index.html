打造良好多终端写作体验

# 首先你要有个博客

推荐用vuepress搭建简单又好看，开源社区（GitHub等）里也有丰富的主题与插件可选

有这几款软件工具你值得拥有👉

# 坚果云

你的博客里应该有个目录用于存放MarkDown文件（即.md文件）

使用坚果云同步文章目录（放文章的文件夹）

# Typora

多平台（Windows、Mac、Linux）十分好用MarkDown编辑器！

支持Latex语法、支持拖拽使用Picgo自动上传图片、表格等等。。。

这里庆祝下typora正式发布啦，已经进入了付费软件行列。

经历了6年的内测，这款优秀的markdown编辑器终于发布了正式版，作者应该是我国人。

# 易码

安卓端MarkDown编辑器。

这软件的开发者老哥酷安，老久没更新了。

用坚果云支持的WebDav方式来同步就好。

这样无论是删改都能即使同步多个终端，也有误删备份

# Mweb

IOS下好用的MarkDown编辑器，支持WebDav同步

还没体验过。

# PicGo

配置好后把图片拖拽到Typora后就帮你自动上传了（你还可以压缩图片后在上传）

这是个图床，用Github当图床挺不错的( •̀ ω •́ )✧

设置CDN后图片访问速度就很不错了。

再图片链接前添加https://cdn.jsdelivr.net/gh/就好。

它也有安卓端，都很好用哈。
