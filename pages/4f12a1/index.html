获取中兴F412光猫超级管理密码

首先开启telnet （控制面板>程序和功能>启用或关闭Windows功能>telnet客户端）

打开cmd （注意，前往不要用Windows Terminal打开，我被他坑惨了）

接着输入telnet 192.168.1.1

如果连不上，如果不让你连

再开一个cmd

一直ping它

ping 192.168.1.1 -l 128 -t

趁它不注意

telnet 192.168.1.1 10128

就可以登陆了

之后会要求你登录

Login: root

password: Zte521

输入 sendcmd 1 DB p DevAuthInfo

一般第一个用户就是的，密码得到了

ps：这时如果你用Windows Terminal打开

你就会遇到这种情况

啥都列出来了，但是第一个用户硬生生被Windows Terminal给吞了（应该是格式化输出有问题，企业特色吗？🤣），别问我是怎么知道的😂，我被这个问题折磨了一天😫

其它命令如下：

修改超级管理员帐号 sendcmd 1 DB set UserInfo 0 Username XXXXXX

修改超级管理员密码 sendcmd 1 DB set UserInfo 0 Password XXXXXX

查看root帐号密码（telnet远程进入的帐号和密码 sendcmd 1 DB p TelnetCfg
