# VuePress部署

# 第一步,下载node.js

去官网下载就行 官网链接: (opens new window) 如果node.js没有添加到环境变量请手动添加

# 第二步,更换镜像源为淘宝源

为啥是淘宝源呢~

因为速度快，嗯(～￣▽￣)～ 就是这样~

命令如下：

#临时：
npm --registry https://registry.npm.taobao.org 
#永久:
npm config set registry https://registry.npm.taobao.org -g
#验证配置是否成功:
npm config get registry

# 第三步，创建文件以及初始化

1、在任意的目录下,创建一个文件夹(名称随意),之后再此文件下创建一个名为 docs 的文件夹

2、在你的随意名字的文件夹下打开终端,使用你喜欢的包管理器进行初始化

3、在docs下创建名为index.md的文件,并在里面写上你的一句话

4、最后启动本地服务器就大功告成了!

注意

记得在package.json中加入以下内容

"scripts": {
    "docs:dev": "vuepress dev docs",
    "docs:build": "vuepress build docs"
  }

    yarn init

    npm init

    // Make sure to add code blocks to your code group

    在此，我推荐使用yarn代替npm

    原因

    因为npm与webpack使用会生成错误的目录

    还有就是yarn 比npm更加方便

    # 后记：

    安装Vuepress一条龙~ 
    npm config set registry https://registry.npm.taobao.org
npm install -g yarn
yarn config set registry https://registry.npm.taobao.org -g
yarn add vuepress

    在package.json中添加脚本:

    "scripts": {
