# VuePress踩坑笔记

# 我踩过的坑

如果你使用yarn docs:dev启动本地服务器报错时

不妨试试cd到docs下打开终端，问题或许就解决了

或者,你是不是忘记再package.json中添加脚本啦😂

# 侧边栏的坑

当你想使用三级侧边栏时，请一定要注意，你必须要有两篇存在，不然是不会显示目录的

sidebar: {
      "/根目录/": [
        {
          title: "大标题1",
          collapsable: true,
         children: [
        { title: "子标题1", path: "文件路径必须存在" },
          { title: "子标题2", path: "文件路径必须存在" },
          ],
        },
        {
          title: "大标题2",
          collapsable: true,
          children: [
            { title: "子标题1", path: "文件路径必须存在" },
            { title: "子标题2", path: "文件路径必须存在" },
          ],
        },
      ],
    },

# 添加脚本的坑

//在使用包管理器时,脚本的添加必须在
"main": "index.js",
//的下方,不然是会报错的😚

# 使用插件的坑

下载好的插件不生效?

原因可能如下:

你可能使用了全局安装vuepress的方式来部署vuepress,

而当你要使用插件时,还需要在你的本地项目里安装vuepress,这样插件才有依赖,它才能正常运行
