    # Windows注册表

    为新建Markdown文件添加右键菜单~ 
    Windows Registry Editor Version 5.00

[HKEY_CLASSES_ROOT\.md]
@="Typora.md"
"Content Type"="text/markdown"
"PerceivedType"="text"

[HKEY_CLASSES_ROOT\.md\ShellNew]
"NullFile"=""
    删除右键菜单可恶的百度云

    找到 [HKEY_CLASSES_ROOT*\shellex\ContextMenuHandlers] 干掉ContextMenuHandlers下的YunShellExt

    找到[HKEY_CLASSES_ROOT\Directory\shellex\ContextMenuHandlers] 也干掉ContextMenuHandlers下的YunShellExt nice ~

    关闭Windows10输入密码时的小眼睛 
    [HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\CredUI]
"DisablePasswordReveal"=dword:00000001
    修改注册表阻止edge浏览器自动升级

    操作如下注册表路径:

    计算机\HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\EdgeUpdate
;新建 DWORD32 键值并重命名为：
DoNotUpdateToEdgeWithChromium
;最后双击该键值修改键值内容：
;空 #代表未定义、不阻止浏览器自动升级
;0  #不阻止浏览器自动升级
;1  #阻止浏览器自动升级
Allowsxs
;值为 1
    20H2卸载自动更新的chrome内核的edge

    使用命令卸载

    rem 路径:
cd C:/Program Files (x86)/Microsoft/Edge/Application/84.0.522.69/Installer 
rem 执行卸载命令 :
setup.exe --uninstall --system-level --verbose-logging --force-uninstall
    添加UAC白名单 
    计算机\HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows NT\CurrentVersion\AppCompatFlags\Layers
;在其下新建字符串值，值的名称是程序的全路径，值数据是“RunAsInvoker”。而且可以用","使用多个设置，比如XP SP3兼容等。
    修复右键菜单的共享不显示 
    [HKEY_CLASSES_ROOT\*\shellex\ContextMenuHandlers\ModernSharing]
@="{e2bf9676-5f8f-435c-97eb-11607a5bedf7}"

    3、改注册表

    # 更多

    在注册表中清除网卡信息： 第一步、定位到

    [HKEY_LOCAL_MCHINE\SYSTEM\CurrentControlSet\Control\Network\{4D36E972-E325-11CE-BFC1-08002BE10318}]

    在该子键下面保存着关于本地连接的信息。在此有一个或多个子键，你可以展开该子键，再选择下面的“Connection”子键，在右侧窗口中将会看到具体是对应着哪个连接，直接将对应以前旧连接的整个子键删除。

    第二步、再删除

    [HKEY_LOCAL_MCHINE\SYSTEM\CurrentControlSet\Services][HKEY_LOCAL_MCHINE\SYSTEM\CurrentControlSet\Services\Tcpip\Parameters\Interfaces]

    下对应的子键即可

    将office 安装到非C盘 
    ;定位到
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion
;修改一下自建的路径
ProgramFilesDir
ProgramFilesDir (x86)
ProgramW6432Dir

    注意!

    自定义路径后,如果使用dism++来清理NEt程序集缓存的话,会导致Excel发生错误
