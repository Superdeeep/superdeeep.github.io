# Windows快捷键

Win+shift+S win10自带截图

Ctrl+c 复制 Ctrl+v 截切 win+D 回到桌面 写给小白的

Shift+F10 鼠标右键

Win+G win10的录屏 其实是xbox的bar😂

Win+. Window10的表情面板，它很可爱的😊

Alt +Tab 多任务切换

Win+B 呼唤出任务栏，全屏下好用
