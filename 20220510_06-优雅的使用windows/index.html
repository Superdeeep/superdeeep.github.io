# 1、禁用几个服务

（1）按下win+r键，输入“services.msc”来打开服务

或者在这里直接搜索“服务”

（2）找到sysmain这个服务(下图是处理好的样子) :::

（3）双击它，然后停止这个服务

接着禁用它


为什么？

因为这个服务特别智障，他会自动在后台帮你打开妮你平时用的软件。初衷是很好，但是谁会每天在同一时间打开一样的软件呢？



按照如上步骤，找到Windows search这个服务，禁用他。

为什么？

因为他会闲时索引你的计算机的文件占用资源，减慢电脑的运行速度。

# 2、启动电源计划的卓越性能。

打开powershell

输入“powercfg -duplicatescheme e9a42b02-d5df-448d-aa00-03f14749eb61”

回车即可（我的powershell改了个样子，这个不用担心，效果都一样）

接着在电源选项中选卓越性能即可（如何打开电源选项? (opens new window) :::）

# 4、谷歌浏览器配置

首先你需要学习如何在谷歌浏览器中添加插件

1、进入设置

2、选择扩展程序

3、打开开发者模式(打开之后，你可以直接将crx文件拖到此处安装插件) :::

4、加载已解压的扩展程序

5、选择文件夹，之后插件就添加好了

几款插件推荐：

AdGuard 广告拦截器 （让你的网页没有广告）

Enable Copy （让不可复制的网页可复制）

Tabliss - A Beautiful New Tab （让你的主页和新建标签页更好看）

Tampermonkey（油猴脚本,让你的知乎和CSDN变得干净无弹窗）

哔哩哔哩助手 [让网页上的哔哩哔哩视频可以下载(可能会被AdGuard屏蔽，下载前请先关闭AdGuard)]

# 5、Win10垃圾清理指南

# 使用磁盘清理工具进行清理

打开此电脑，右键C盘























选择属性

















选择磁盘清理

















点击确定即可清理

















想要清理更多,请点击清理系统文件

















点击确定即可

意外情况：

若出现提示称，

清理更新后将无法还原到以前的更新

直接点确定即可

只要不乱搞，win10蓝屏是自己会修复的

特殊情况除外

# 使用360清理大师-Windows10进行清理

•说明，经过本人使用这款360产品为良心之作，大可放心使用

•绝对没有广告以及全家桶行为

下载链接 (opens new window)

打开软件 点击快速扫描

等待完成。。。

接着在列出的列表中，把有关（系统，常用软件垃圾）的内容全部选上⚆_⚆

注意：

在开机可加速中，千万不要把MemReduct选上了，不然之前咱们之前的内存清理软件开机之后就不会自动启动了（感觉就像不见了）

注意：

有时候他会默认勾选几个软件卸载，清理的时候注意下。这只是个例

然后点击清理所有

退出软件时，请点击任务栏的电池电量（有时候是内存，有时候是温度，看着是这个地方就像）那个地方退出，不然是会占用系统资源的。

接着我们在任务管理器中将360清理大师默认启动给它关掉

右键任务栏》》打开任务管理器》》选择启动选项》》找到360清理大师Tray模块》》在右下方点击禁用

# [高级清理]使用dism++ 进行清理

下载链接 (opens new window)

注意，dism++有的选项与系统内部相关，请不要乱搞，不然搞坏了开不了机就...(/ω＼)

如果弹出什么窗口点击继续或者确定就行。

等待清理完成就OK了

# 结语：

如何让电脑使用更加舒适？其实对于Windows来说，下载一些良心的无广告软件，保持良好的使用习惯即可。

----(｡･∀･)ﾉﾞ小经验----

•软件不要老安到C盘

•下载软件去官网下载

•各种下载站的软件不要乱下（最好能不下就不下），要下请选择普通下载，压缩格式一般可以，exe大多是捆绑软件或是病毒

•360杀毒调整好了日常使用很安静的，占用也很小，就是比较喜欢删exe，这是没办法的，毕竟防毒比较厉害。

•火绒比360更安静，默认就是很好的（但是杀毒比360弱）

•要是真的来新的强悍病毒，没有杀毒软件能挡住，只有中招之后进行云防控。所以重要文件要备份好（最好有俩备份）
