# 4、bubm数码配件收纳包 (opens new window)

还在为U盘多难以收纳，担心它掉而烦恼吗？还在为耳机在口袋里打结而不想听歌吗？

有了它，这些统统走开！

它的特点是弹簧开口，可以单手开合，但闭合时却又十分紧闭，完全不用担心里面的东西会掉出来。小小一只放哪都方便，风格也很简洁，喜欢就入了吧。

# 5、eVtran USB3.0银灿IS903 32G SLC 高速U盘 (opens new window)

SLC的芯片很耐读写，U盘经常存储数据推荐选它。本人目前有就一个，至今没翻车，速度也没掉，闪存时东芝的32纳米，怎么来说都挺行的，就算芯片时仓库货质量依旧。当然毕竟不是大厂可能偶尔有个漏网之鱼的次品也说不准哈。

# 6、闪迪U盘 128g USB3.2 CZ880u盘 (opens new window)

mlc芯片固态读写，加上是大企业生产，质量自然没得说，价格也十分冻人。但是终身质保非常nice，记得U盘身上的字别搞掉了，不然是不会保的😛。

话说闪迪的U盘都挺不错的，反正我手上最近1-2年买的这么多闪迪的U盘都没坏过，其中一个U盘更是被我反复做系统盘，格式化了不下数十次，写入了居多数据，愣是没坏。😉

# 7、Mr.green德国折叠指甲刀 (opens new window)

超级小的指甲刀，小到挂在钥匙串上没有任何问题😉。即便如此它也十分锋利，剪起指甲嘎嘎脆！

# 8、xmyb 简约便携羊皮超软小钱包 (opens new window)

特别特别小，放进口袋刚刚好，及不占地方有收纳了零钱，各种各样的钥匙和小零碎，简约好看ヾ(≧▽≦*)o，价格也十分友好(＾Ｕ＾)ノ~ＹO

# 9、植护香水洗衣液香味持久留香家用实惠整箱批护理机洗专用深层洁净 (opens new window)

价格是推荐的主要原因，耐用又实惠😊，洗完了会有淡淡的香味哦。家用的话还是蓝月亮，毕竟姜还是老的辣 (￣▽￣)
